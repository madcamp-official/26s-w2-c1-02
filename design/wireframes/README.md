# Rehearsal.io UI 와이어프레임 (SVG)

기능명세서(Google Sheets) 기반, 토스 레이아웃 문법 + `#FFAA00` 주황 톤.
Figma REST API는 쓰기가 불가능하므로 SVG로 제작 → Figma에 드래그하면 편집 가능한 레이어/텍스트로 변환됩니다.

## Figma에 가져가기

1. Figma에서 대상 페이지 열기
2. Finder에서 이 폴더의 SVG 파일을 **캔버스로 드래그&드롭**
   - 한 번에: `rehearsal-wireframes-all.svg` (10개 그룹 · 28개 화면, 그룹 간 간격 300px 포함)
   - 그룹별로: `01-auth.svg` ~ `10-common.svg`
3. 드롭 후 텍스트·도형 모두 개별 레이어로 편집 가능 (폰트는 Inter로 매핑됨)

## 그룹 구성 (28화면)

| 파일 | 그룹 | 화면 |
|---|---|---|
| 01-auth | 인증 | 로그인 / 회원가입(이메일 인증) / 아이디·비밀번호 찾기(관리자 문의) |
| 02-main | 메인 | 팀 블록 리스트 + 팀 추가 + 마이페이지 진입 |
| 03-team | 팀 | 팀 만들기(이메일·링크 초대) / 초대 수락 / 팀 페이지 / 팀 관리 바텀시트(나가기·삭제) |
| 04-prep | 발표 준비 | 발표 설정(PDF 선택 업로드·진행방식·페르소나 5종 중복선택·질의수·제한시간) / 전처리 상태 3종 / 녹음 파일 업로드 |
| 05-present | 발표 | 발표중(슬라이드·타이머·초과표시) / STT 변환 로딩(+실패) / 질의응답 전환 확인 |
| 06-qna | 질의응답 | 질문 TTS 재생(근거 배지·다시 듣기) / 답변 자동 녹음(패스·제한시간) / 꼬리질문 / 종료·리포트 자동 생성 |
| 07-history | 이전 발표 | 상세-스크립트 탭 / 상세-Q&A 로그 탭(페르소나·패스·꼬리표시) / 삭제 확인 |
| 08-report | 분석 | 단일 리포트(강약점·발표 습관) / 성장 리포트(회차 비교) |
| 09-mypage | 마이페이지 | 계정 관리 / 비밀번호 변경 / 탈퇴 확인 |
| 10-common | 공통 | 마이크 권한(요청·거부 안내) / API 오류·이탈 경고 |

## 반영된 결정사항 (2026-07-10 확정)

- 계층: 팀 > 세션(발표 1회) 2계층, 회차 비교는 성장 리포트에서 처리
- 질문 근거 표시 O / 녹음 파일 업로드 경로 O / TTS 후 답변 자동 녹음 O / 리포트 자동 생성 O
