핵심 루프 (한 세션의 일생)
  1. PDF 업로드 → material.py 파싱 → slides.json
  2. 녹음 업로드 → stt_queue → transcripts.segments(초 float)
  3. POST /qna/generate → qna_jobs: slides+transcript로 LLM 질문 생성 → 질문별 TTS wav 저장
  4. 답변 업로드 → 답변 STT → follow_up() 판정 → 꼬리질문(부모 persona 승계, depth≤1)
  5. qna/end → 리포트 잡: compute_speaking_metrics(숫자) + generate_report(LLM 채점) — 여기서 HTTP 라우트만 아직 없음
